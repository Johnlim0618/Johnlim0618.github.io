Windows Event Investigation Practice

This page includes a small PowerShell script that filters common Windows Security Event IDs useful for beginner investigation practice.

What this project shows

  • Authentication event review
  • Privileged group change visibility
  • PowerShell-friendly log filtering

Example artifact

$ids = 4624,4625,4732
Get-WinEvent -LogName Security |
Where-Object { $ids -contains $_.Id } |
Select-Object TimeCreated, Id, ProviderName, Message |
Format-Table -AutoSize
